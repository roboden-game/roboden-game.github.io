1. What information does the Application obtain and how is it used?

1.1. User Provided Information

The Application does not require any user registration. The Application does not require the user to provide any personal information. The only User Provided Data that this Application uses is the information you provide directly within the Application: when player wishes to share their highscores, they're asked to set their username.

The game save states are only stored locally on mobile device, within the Application private folder.

1.2. Automatically Collected Information

The Application does not collect any information automatically. The Application does not share any information automatically with third parties.

Does the Application collect precise real time location information of the device? The Application does not collect any location information.
Do third parties see and/or have access to information obtained by the Application? Not applicable since the Application does not collect any information automatically.
What are my opt-out rights? You can neven set any username and avoid publishing any of your highscores.

2. Data Retention Policy, Managing Your Information

This Application does not automatically collect, share nor retain any information.

If user decides to publish their highscores, this data will be sent to the server and may become part of the public leaderboard. The leaderboard entry includes this information: username (as specified by the user), their score and completion time.

It's impossible to delete a leaderboard entry when it's published since we don't know who it belongs to. There is no registration or authorization, so anyone can send any data associated with username. This abstract username should not be treated as reliable identifier.
